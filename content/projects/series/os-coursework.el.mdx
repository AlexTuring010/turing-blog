Τρεις εργασίες, ένα εξάμηνο, αρκετά segfaults για μια ζωή.

## Τι είχε

Η σειρά εργασιών στο μάθημα Λειτουργικών Συστημάτων ζητούσε να γράψουμε *κάτω* από το user-space που ξέραμε. Δεν να καλέσουμε threads — να τα φτιάξουμε. Δεν να βλέπουμε virtual memory — να γράψουμε τον page replacement αλγόριθμο. Δεν να χρησιμοποιήσουμε filesystem — να εκθέσουμε ένα.

Καθεμία έχει το δικό της case study.
