Threads από user-space. Ένα scheduler που κάνει round-robin. Mutexes. Condition variables που λειτουργούν, νομίζω.

## Τι ζητούσε

Να γράψουμε μια POSIX-style thread library που να μη χρησιμοποιεί `pthread`. Όλα τα tools που είχαμε στη διάθεσή μας: `setjmp/longjmp`, `ucontext`, `signal`. Καμία βοήθεια από το kernel.

Το hard part δεν ήταν το context switch. Ήταν να καταλάβω *γιατί* στο C είναι αποδεκτό να μη ξέρεις τι θα γίνει αν επιστρέψεις από μια setjmp/longjmp σε μια ξεκάθαρη περίπτωση race condition.
