Threads from user-space. A round-robin scheduler. Mutexes. Condition variables that work, I think.

## What it asked for

Write a POSIX-style thread library without using `pthread`. Tools we had: `setjmp/longjmp`, `ucontext`, `signal`. Zero help from the kernel.

The hard part wasn't the context switch. It was figuring out *why* C considers it acceptable to leave undefined what happens if you return from a setjmp/longjmp in a clean race-condition case.
